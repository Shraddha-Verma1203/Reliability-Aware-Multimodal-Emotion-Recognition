# MER-Reliability-Fusion
# Reliability-Aware Multimodal Emotion Recognition

## Overview
A multimodal emotion recognition system that combines Text, Audio, and Facial Features using Reliability-Aware Adaptive Fusion.

## Features
- Text Emotion Recognition
- Audio Emotion Recognition
- Facial Emotion Recognition
- Reliability Scoring Module
- Adaptive Fusion Mechanism
- Conflict Detection
- Explainable Predictions
- Streamlit Dashboard
